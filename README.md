# DL for ECG Recog
 The classification of electrocardiogram signals

## Environment
Python 3.7  
tensorflow 2.1.0 GPU  
scikit-learn 0.22.1  

## dataset
you can download the data files [here](https://tianchi.aliyun.com/dataset/dataDetail?dataId=94490)

## Tips
1. run file data_process.py
